# UFO Shooting Game from you childhood
Made with the raylib library for C++

## Prerequisites
1. C++ compiler
2. Raylib library (Download link : https://www.raylib.com/)

## Game Controls
1. W : Move ship up
2. S : Move ship down
3. ENTER : Fire bullets

## Game Rules
1. Hit the enemies (Purple) with bullets
2. Don't hit the bombs (Red), or they will explode and kill you
3. You die if any entity hits you directly
